Python Data Structures Handbook


Lists

Lists can be created as both populated lists and as empty lists.
Empty lists are created by the name of the list, equals signs, and a pair of braces.
1. An empty list
E.g. fruits = [ ]
2. A populated list
E.g. fruits = ["apples", "bananas", "cranberries", "kiwis", "melons"]
Lists in python have various methods for accessing and manipulating the data based on your needs.
Before we dive deep in the methods, lets first introduce another data structure known as tuples.



Tuples

Tuples are somewhat similar to lists with a few key differences.
Tuples can also be created as both populated and empty tuples.
Empty tuples are created by the use of brackets.
They're created by the name of the tuple, equals sign, and a pair of brackets.
1. An empty tuple
E.g. fruits = ()
2. A populated tuple
E.g. fruits = ("apples", "bananas", "cranberries", "kiwis", "melons")
Tuples are immutable. This means that once a tuple is created its data cannot be changed.
This is the key difference between a list and a tuple.


A list is mutable while a tuple is not.
Following this fact, a list has more methods available than a tuple has.
There are a few common methods that are available to both lists and tuples.

Some few methods available to both tuples and lists are:


1. len(). This returns the number of elements in the list or tuple. I.e. the length of the list/tuple
e.g. len(fruits)
2. index(). This returns the position of the element in the list or tuple.
e.g. fruits.index("melons")
3. count(). This returns the number of similar elements in the list or tuple.
e.g. fruits.count("apples")

Methods available to lists


1. append(). This adds another element to the end of the list.
e.g fruits.append("strawberries")
2. clear(). This deletes all the elements in the list and returns an empty list.
e.g fruits.clear()
3. copy(). This copies the elements of one list into another list.
e.g fruits1 = fruits.copy()
4. extend(). This adds the contents of another list into the list.
e.g fruits.extend(fruits1)
5. insert(). This adds another element into a specified index of the list.
e.g fruits.insert(3, "pawpaws")
6. pop(). This removes the last element of the list.
e.g fruits.pop()
7. remove(). This removes a specified element from the list.
e.g fruits.remove("apples")
8. reverse(). This reverses the order of the list.
e.g fruits.reverse()
9. sort(). This rearranges the elements of the list.
e.g fruits.sort()

Tuples occupy less memory as compared to lists due to the fact that they are immutable and un-dynamic.
They also take less time as compared to lists of the same size.

Sets

Sets are used to store data in which order is not of importance. Sets can store different types of data e.g. strings, integers, floats, Booleans.
Sets do not store duplicate data.
Sets can be initialized by the use as empty sets or populated sets.
Empty sets are created by name of the set, equals signs, the name set followed by a pair of brackets.
E.g. details = set ()

Methods available to sets

1. add(). Adds another element into the set.
e.g details.add("name")
2. copy(). Copies the contents of one set into another set.
e.g details1 = details.copy()
3. clear(). This deletes all the elements in the set and returns an empty set.
e.g details.clear()
4. len(). This returns the number of elements in the set. returns the length of the set.
e.g len(details)
5. remove(). This removes an element from the set.
e.g details.remove("name")
6. discard(). This removes a specified element from the set but if the element does not exist it does nothing(does not return an error)
e.g details.discard("name")

kenyan = set (["max", "travis", "scot", "amber", "crystal"])
french = set (["trois", "magret", "amber", "chateau", "bear"])
spanish = set (["alejandro", "pendejo", "max", "bear", "katalia"])
mixed = set (["max", "amber"])

7. union(). This joins two sets and returns them as one.
e.g kenyan.union(french)
8. intersection(). This returns only the elements present in both sets.
e.g kenyan.intersection(spanish)
9. difference(). This returns the elements which are are present in either sets but not in both.
e.g kenyan.difference(french)
10. in. returns a boolean to the query is the element is present or absent.
e.g "max" in kenyan
11. not in. returns a boolean to the query if the element is present or absent.
e.g "max" not in kenyan
12. issuperset(). Returns a boolean if all some of the elements in the set are all contained in the other set.
e.g kenyan.issuperset(mixed)
13. issubset(). Returns a boolean if all the elements in the set are contained in the other set.
e.g mixed.issubset(kenyan)
14. pop(). Removes the last element of the set.
e.g details.pop()


Dictionaries/Associative arrays/Maps

Data dictionaries are data structures used to store associated data. They contain key-value pairs.
The input is termed as the key while the output is termed as the value.
The dictionaries map one value to another. Dictionaries do not store duplicate data.
There are several ways of initializing dictionaries:
1. An empty dictionary can be intitialized by the use of the dictionary name, equals sign, and a pair of curly braces
e.g employee = {}
2. Can also be initialized by the use of dictionary name, equals sign, the name 'dict', and a pair of brackets
e.g employee = dict()
A data dictionary can store various data types of which the order does not matter.
a) employee = {"name": "Lucas", "age": 23, "gender": "Male"}
b) employee = dict(name = "Lucas", age: 23, gender: "Male")

Methods available to dictionaries


1. len(). Returns the number of key-value pairs in the dictionary
e.g len(employee)
2. clear(). Deletes all the key-value pairs in the dictionary and returns an empty dictionary
e.g employee.clear()
3. pop(). Removes the a specified key-value pair in the dictionary.
e.g employee.pop("name")
4. popitem(). Removes the last key-value pair in the dictionary.
e.g employee.popitem()
5. copy(). Copies data from one dictionary to another.
e.g employee1 = employee.copy()
6. keys(). Returns all the keys present in the dictionary.
e.g employee.keys()
7. values(). Returns all the values present in the dictionary.
e.g employee.values()
8. get(). Returns the value for a given key.
e.g employee.get("age")
9. items(). Returns all the key-value pairs.
e.g employee.items()
